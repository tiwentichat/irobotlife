# 入门级：从入门到熟练使用


好的问题，能带来好的答案。所以我们需要学会提问

在面对AI的时候，我们也可以练习一下我们的提问技巧。

Prompts，一般翻译为提示词，不过在我看来，我们与AI的互动更像：

我们提问题、提要求、提任务

AI回答问题、按要求反馈结果（比如让它按照格式帮我们写故事、写代码）、执行任务反馈执行结果（比如翻译）
